📄 Cerrebro Tutorials ERP: Project Description
Project Title: Cerrebro Tutorials - Next-Generation Institute Management ERP

Developed By:  Mr. Sahil Thonge

Overview: Cerrebro Tutorials ERP is a comprehensive, full-stack School and Institute Management System built on the MERN stack. Designed to handle the complex daily operations of educational institutes, it seamlessly connects Admins, Teachers, and Students through a real-time, glassmorphic UI. Moving beyond basic data entry, this ERP features "Selective Academic Year Scoping" for decade-long data integrity, a real-time communication engine, and a dedicated AI Tutor built directly into the student dashboard.

Key Features & Modules:

👑 Admin Finance & Analytics Engine:

Global Academic Year filtering (e.g., viewing data strictly for "2025-26").

Automated financial ledgers tracking Total Income (Fee Collection), Expenses (Rent, Salaries), and Net Profit.

Real-time tracking of online student payments with CSV export capabilities.

👨‍🏫 Advanced Teacher Workspace:

Custom dashboard with quick actions for assignments, syllabus, and marks entry.

Visual "My Attendance" calendar for teachers to track their own presence/leaves.

Frictionless student attendance marking system with bulk actions and UI-color-coded states.

🎓 Student Hub & AI Integration:

Floating "Dynamic Island" widget offering quick access to human teachers or the AI Tutor.

Cerrebro AI: A custom-prompted, blazing-fast tutor powered by Groq (Llama-3.3-70b-versatile), specifically calibrated to guide 8th-12th grade students through Math and Science step-by-step.

⚡ Real-Time Infrastructure:

Socket.io-powered live chat system between students and teachers.

Instant push-notification system with a professional dropdown UI and "Mark all as read" capabilities.

Tech Stack:

Frontend: React.js, Tailwind CSS (Custom Slate/Indigo/Teal themes), React Router, React Hot Toast.

Backend: Node.js, Express.js.

Database: MongoDB with Mongoose (Strict schema design).

AI Integration: Groq SDK (Llama 3.3).

Real-time: Socket.io.
